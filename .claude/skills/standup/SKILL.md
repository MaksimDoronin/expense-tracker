Собери standup-отчёт за последние 24 часа на основе коммитов в репозитории.

## Шаги

1. Получи коммиты за последние 24 часа:

```bash
git log --since="24 hours ago" --format="%H|%ai|%s|%b" --all
```

2. Если коммитов нет — сообщи: «За последние 24 часа коммитов не найдено.»

3. Для каждого коммита:
   - Извлеки `type(scope)` из заголовка (Conventional Commits).
   - Кратко опиши суть изменения своими словами (1–2 предложения), опираясь на заголовок и тело коммита.
   - Зафикси точное время коммита.

4. Отсортируй коммиты от самого раннего к самому позднему (по дате).

## Формат вывода

```
## Standup — <дата сегодня>

### Что было сделано

**<ЧЧ:ММ>** · `<type>(<scope>)`
<Описание что сделано, 1–2 предложения>

**<ЧЧ:ММ>** · `<type>(<scope>)`
<Описание>

---
Итого коммитов: <N>
```

## Правила

- Время отображать в локальном формате `ЧЧ:ММ` (без секунд).
- Описание — на **русском языке**, понятным языком без технического жаргона там, где это возможно.
- Если у коммита нет scope — показывай только `type`.
- Не включай `Co-Authored-By` строки в описание.
- Если несколько коммитов посвящены одной задаче — можно сгруппировать их под одним пунктом с пометкой `(N коммита)`.
