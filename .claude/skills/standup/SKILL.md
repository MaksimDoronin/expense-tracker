Собери standup-отчёт за последние 24 часа на основе коммитов в репозитории.

## Алгоритм работы скилла

1. Получить текущую дату (date)
2. Взять коммиты за вчера: git log --since="yesterday" --until="today" --oneline
3. Для каждого коммита получить детали через git show
4. Сформировать связный текст: что было сделано, в каких модулях
5. Вывести отчёт в формате: «Вчера занимался X, Y, Z»

## Пример вывода


Вчера занимался документированием проекта:
- подключил Swagger, все API-эндпоинты доступны в браузере
- покрыл декораторами все контроллеры модуля транзакций
- добавил JSDoc ко всем публичным методам
- удалил заглушки AppController/AppService
- написал README с инструкцией запуска
- добавил скилл для автоматических коммитов
